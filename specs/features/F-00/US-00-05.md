# US-00-05 · Day-Ahead Market Prices (D-04)

## User Story

**As a** developer building the PoC data layer, **I want to** provide hourly day-ahead electricity prices for the next 24 hours from EPEX SPOT NL, **so that** the Market Forecast card can render the price chart and the AI recommendation logic has price input data.

## Description

Day-Ahead Market Prices contains hourly electricity prices for the next 24 hours as published by EPEX SPOT for the Netherlands bidding zone. This series drives the price chart in the Market Forecast card and is the primary input to the AI recommendation logic. Stored in `market_forecast_prices.json`.

The dataset contains: market and bidding zone name (`market`), currency (`currency`), generation timestamp (`generated_at`), and an array of 24 hourly price objects. Each price object has: start of the pricing hour (`hour_start`) and day-ahead clearing price (`price_eur_mwh`).

A realistic April weekday shape should include: an evening peak around 17:00–19:00 (~150–170 €/MWh), a deep overnight trough around 02:00–05:00 (~15–25 €/MWh), and a midday solar suppression dip around 11:00–14:00 (~55–80 €/MWh). The spread between overnight low and evening peak typically reaches 7–10× in spring.

## Acceptance Criteria

- **AC-1**: The dataset contains a `market` field set to "EPEX SPOT NL".
- **AC-2**: The dataset contains a `currency` field set to "EUR".
- **AC-3**: The dataset contains a `generated_at` timestamp in UTC-offset format.
- **AC-4**: The `prices` array contains exactly 24 hourly price objects spanning a continuous 24-hour period.
- **AC-5**: Each price object contains `hour_start` (timestamp) and `price_eur_mwh` (numeric).
- **AC-6**: Price hours are spaced exactly 1 hour apart with no gaps or duplicates.
- **AC-7**: The price curve includes an evening peak (~150–170 €/MWh around 17:00–19:00).
- **AC-8**: The price curve includes a deep overnight trough (~15–25 €/MWh around 02:00–05:00).
- **AC-9**: The price curve includes a midday solar suppression dip (~55–80 €/MWh around 11:00–14:00).
- **AC-10**: The spread between the lowest and highest price reaches 7–10× (realistic spring NL market).
- **AC-11**: All price values are positive numbers expressed in €/MWh.
