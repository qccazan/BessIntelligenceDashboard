# US-04-02 · Operational State and Real-Time Power

## User Story

**As a** operator verifying that a battery is behaving as expected, **I want to** see the selected battery's current operational state and real-time power output, **so that** I can confirm the asset is in the correct mode and delivering the expected power.

## Description

The Current State card displays the operational mode (Charging / Discharging / Idle / Fault) as a colour-coded pill, alongside the real-time power reading in kW. Power is signed: positive for charging, negative for discharging, zero for idle. A directional label accompanies the power value.

## Acceptance Criteria

- **AC-1**: An operational state pill is displayed in the card header, with one of four values: Charging, Discharging, Idle, or Fault.
- **AC-2**: The state pill colour is consistent with the colour scheme used in the Fleet Overview table.
- **AC-3**: The real-time power value is displayed with a unit (kW) and a directional label.
- **AC-4**: Charging assets display a positive kW value; discharging assets display a negative kW value; idle assets display ~0 kW.
- **AC-5**: The state pill and power sign are mutually consistent for every selectable asset.
